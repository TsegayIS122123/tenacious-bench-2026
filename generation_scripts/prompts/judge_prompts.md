# Judge Prompts for Tenacious-Bench LLM-as-Judge

## Dimension 1: input_coherence

**Prompt:**
Rate the following task from 1-5 on INPUT COHERENCE.
1 = incoherent/contradictory, 3 = makes sense but has minor issues, 5 = perfectly coherent.

Task: {task_json}

Respond with: Score: X - Reason: [brief explanation]

text

**Calibration:**
- Score 1: Missing required fields, illogical combinations, nonsense values
- Score 3: Has all fields but some minor inconsistencies
- Score 5: All fields present and logically consistent

## Dimension 2: ground_truth_verifiability

**Prompt:**
Rate the following task from 1-5 on GROUND TRUTH VERIFIABILITY.
1 = cannot be verified by script, 3 = partially verifiable, 5 = fully machine-verifiable.

Ground truth: {ground_truth_json}
Rubric: {rubric_json}

Respond with: Score: X - Reason: [brief explanation]

text

**Calibration:**
- Score 1: Subjective criteria only ("sounds good", "feels right")
- Score 3: Mix of mechanical and subjective checks
- Score 5: Binary checks, banned phrase lists, regex patterns only

## Dimension 3: rubric_application_clarity

**Prompt:**
Rate the following rubric from 1-5 on APPLICATION CLARITY.
1 = ambiguous/can't apply, 3 = somewhat clear but has ambiguity,
5 = clear, mechanical, can be applied consistently.

Rubric: {rubric_json}

Respond with: Score: X - Reason: [brief explanation]

text

**Calibration:**
- Score 1: "The email should be good" (no mechanical criteria)
- Score 3: Has weights but ambiguous scoring guidance
- Score 5: Each dimension has explicit scoring_type, banned_phrases, or required_elements

## Thresholds by Source Mode

| Source Mode | input_coherence | ground_truth_verifiability | rubric_clarity |
|-------------|-----------------|---------------------------|----------------|
| trace-derived | ≥4 | ≥4 | ≥4 |
| programmatic | ≥4 | ≥4 | ≥3 |
| multi-llm-synthesis | ≥3 | ≥4 | ≥3 |
| hand-adversarial | ≥3 | ≥3 | ≥4 |

## Model Rotation (Preference Leakage Prevention)

- If generated by Claude → judge with GPT-4
- If generated by GPT-4 → judge with Claude
- If generated by Qwen → judge with DeepSeek
